## Nuowaxingyun

This module will allow you to control the following Nuowaxingyun media server products: RX6.

### Configuration

- Enter the IP address of the product in the configuration settings.

**Available actions:**

- Program
- Pause_program
- Play_program
- Open_ftb
- Close_ftb
- Open_volume
- Close_volume
- ppt_pgup
- ppt_pgdn
- Stop_program
- Volume_up
- Volume_down
